# 📦 Installation

## Requirements

- Node.js
- npm

### Install Dependencies

```bash
npm install
```

### Configure Environment

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
